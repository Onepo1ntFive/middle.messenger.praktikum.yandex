## Установка

- `npm run build` - запуск сборки проекта 
- `npm run start` - запуск локального сервера для разработки. Проект будет доступен по адресу `localhos:3000`

## Использование

На данном этапе реализованы статические страницы. Для перехода между доступными страницами предусмотрено временное меню в правой части экрана.
